# Security Policy

## Supported Versions

We aim to support the latest main branch and the most recent production release.

## Reporting a Vulnerability

Please report security issues via GitHub Security Advisories:
https://github.com/antonin686/wuwapal/security/advisories

If you cannot use GitHub advisories, reach out via the WuwaPal Discord server
and request a private contact channel: https://discord.gg/DFKG4nqUD4.

Please do not open public issues for security reports. We will acknowledge
receipt within 72 hours and provide a status update once triaged.
